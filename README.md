# LeetCode Challenge

My Java solutions to LeetCode problems, written while practicing data structures and algorithms.

## Structure

- `src/` — solution source files, one per problem
- `lib/` — any external dependencies (jars)
- `bin/` — compiled output (not tracked in git)

## Running a solution

```
javac -d bin src/App.java
java -cp bin App
```
